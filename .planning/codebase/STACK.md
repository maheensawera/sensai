# Tech Stack

## Core Technologies
- **Framework:** Next.js 15.2.8 (App Router)
- **UI Library:** React 19.2.4
- **Language:** JavaScript/Node.js

## Styling & UI Components
- **CSS Framework:** Tailwind CSS 4.2.1
- **Component Library:** Radix UI primitives with Tailwind
- **Animations:** tailwindcss-animate, tw-animate-css
- **Icons:** Lucide React
- **Markdown Rendering:** @uiw/react-markdown-preview, @uiw/react-md-editor
- **Data Visualization:** Recharts

## Data & State Management
- **Database ORM:** Prisma 5.10.2
- **Database:** PostgreSQL (via Prisma datasource)
- **Form Handling:** react-hook-form
- **Schema Validation:** Zod

## Background Jobs & AI
- **Background Workflows:** Inngest 3.44.3
- **Generative AI:** Google Generative AI (@google/generative-ai)

## Authentication & Utilities
- **Authentication:** Clerk (@clerk/nextjs 6.38.3)
- **PDF Generation:** html2pdf.js
- **Date Formatting:** date-fns
